  2. Exported the env from macOS and tried to import on Linux

    libcxxabi=4.0.1

    Possible issue with pip:
    pip uninstall setuptools
    conda install -c anaconda setuptools

  3. Linux: Check Disk Space

    4. df command – Show the amount of disk space used and available on Linux file systems.
    du command – Display the amount of disk space used by the specified files and for each subdirectory.
    btrfs fi df /device/ – Show disk space usage information for a btrfs based mount point/file system.

  4. DLIB Issue #1: image not found

    5. Solution:
    Install from source (10 mins)
    pip install 'git+https://github.com/davisking/dlib.git’

  5. Remote Access to IPython Notebooks via SSH

    6. On the remote machine, start the IPython notebooks server:
    ipython notebook --no-browser --port=8889.

    On the local machine, start an SSH tunnel:
    ssh -N -f -L localhost:8888:localhost:8889 baiqiang@turku.silo.ai .

    Now open your browser on the local machine and type in the address bar:
    localhost:8888
